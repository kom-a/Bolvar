# Bolvar
![Demo](Bolvar/Res/demo.png)

Bolvar is free and open source find and repalce tool.
## Features
* Path validation
* Masks for searching and excluding files
* Two implementations
    * Just searching
    * Replacing
* Preview found matches
* Background working
* Cancellation
